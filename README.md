# PythonProject
 A Hangman Game built using Python and Tkinter. It includes a welcome screen, difficulty levels, word categories, and a score system. Players guess letters using the keyboard, track scores across rounds, and enjoy a user-friendly interface, making it ideal for academic projects and beginners.

 **Features**
Welcome screen with background image
Difficulty levels: Easy, Medium, Hard
Multiple word categories
Keyboard-based letter guessing
Score tracking system
User-friendly GUI

 **How to Run**
Install Python (3.x)
Install required library: pip install pillow
Run the file: python hangman.py
